SVG

SVG står for scalable vector graphics. Det er et 2D vektorbasert bildeformat som baserer seg på XML. SVG har svært god skalering, og man kan forstørre og forminske bildet uten at det endrer kvaliteten. Den kan manipuleres med CSS eller med JavaScript. SVG blir også anbefalt av W3C.

SVG er fint å bruke når man ønsker en enkel illustrasjon, som for eksempel en logo, ikoner, enkle animasjoner, skjemaer og grafer. Det finnes en del standardfigurer som man kan tegne, som bokser, sirkler, ellipser og tekst. Man kan også legge til ulike effekter slik at figuren man lager blir en liten animasjon.
